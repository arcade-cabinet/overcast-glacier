# Branding & Visual Identity

## Color Palette (Tailwind)
The visual language is "Midnight Arctic Glitch".

| Color Name | Hex | Usage |
| :--- | :--- | :--- |
| **Primary** | `#0F172A` | Deep midnight blue background, the "Overcast" sky. |
| **Accent Ice** | `#7DD3FC` | Bright cyan glow, UI highlights, Matrix code. |
| **Snow White** | `#F8FAFC` | Snow, text, primary UI elements. |
| **Glacier Blue** | `#1E40AF` | Shadows, deep ice, secondary backgrounds. |
| **Frost Highlight** | `#60A5FA` | Active states, ability glows. |
| **Warning Red** | `#EF4444` | Low warmth, enemy attacks, danger. |
| **Hot Cocoa** | `#8B4513` | Collectibles, warmth restoration. |
| **Matrix Green** | `#10B981` | Glitch effects, "Hacked" status (tinted blue mostly). |

## Typography
*   **Headings / UI Titles:** `Orbitron` (Weights: 400, 700) - Futuristic, Matrix feel.
*   **Body / Flip Phone:** `VT323` (Monospace Pixel) - Retro tech, terminal text.
*   **Accents / Fun:** `Luckiest Guy` - Playful, round, "gamey" feel.

## UI Aesthetic
*   **Frosted Glass:** Backdrop blur, white/10 opacity, thin cyan borders.
*   **Y2K Tech:** Flip phones, chunky digital cameras, pixelated icons.
*   **Motion:** Everything has weight. Screen shake on impact, slow-mo on kills.
*   **Particles:** Heavy snow, digital "code rain" shards, steam from cocoa.

## Logo
*   **Text:** "Overcast: Glaciers!" in Orbitron, white with cyan glow shadow.
*   **Icon:** Kung-fu kitten silhouette (mid-kick) on a jagged glacier peak.
